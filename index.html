A signal based frontend library with fine-grained reactivity

Prelude lets you develop web application in a familiar component-based functional style

Learn More API Reference

Develop using JavaScript and TypeScript

Prelude works directly in the browser without build process using just JavaScript or use a bundler like Vite for complex applications

import {h, signal, render} from 'https://esm.sh/wrnrlr/prelude'

function App() {
  const count = signal(1)
  return [
    h('button', {onClick:e => count(i=>i-1)}, '-'),
    h('span', count),
    h('button', {onClick:e => count(i=>i+1)}, '+')
  ]
}

render(App, document.body)
